# C++ Zahlenraten Spiel

---

## Beschreibung

Dieses Projekt ist ein kleines Zahlenraten-Spiel, das ich im Rahmen eines Schulprojekts entwickelt habe.  
Der Benutzer muss eine vorgegebene Zahl erraten und bekommt Hinweise.

---

## Funktionen

- Eingabe von Zahlen
- Vergleich mit einer festen Zahl
- Hinweise bei falschen Eingaben
- Wiederholung bis zur richtigen Lösung

---

## Was ich gelernt habe

- Nutzung von Schleifen (while)
- Arbeiten mit Bedingungen
- einfache Benutzerinteraktion
- grundlegende Spiellogik
